## Asthma Disease Data Analysis

### Description
Exploratory data analysis of a clinical dataset to identify 
key risk factors associated with asthma.

### Tools
R · tidyverse · dplyr · ggplot2

### What I did
- Cleaned and transformed clinical data
- Analyzed key risk factors: smoking, BMI, allergies, lung function
- Built 10+ visualizations: heatmaps, boxplots, bar charts

### Key Findings
- Smoking and high BMI are the strongest risk factors
- Strong correlation between allergies and asthma prevalence
